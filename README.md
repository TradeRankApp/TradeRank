# TradeRank

Verified trading performance. Real leaderboards. No fake gurus.

> "Anyone can post a winning trade. TradeRank shows the full picture."

## Deploy

1. Push this repo to GitHub
2. Import to vercel.com
3. Framework: Vite — build command: `npm run build` — output: `dist`
4. Deploy

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:5173
